## 🗺️ What is Kataho?

**Kataho** is Nepal's first precision digital addressing platform — dividing the entire country into **15.2 billion micro-grids**, each just **3.5 × 2.8 meters** in size. Every grid gets a unique, pronounceable code built from everyday Nepali words.

> Share any location in Nepal in under **3 seconds** — home, river, field, or landmark.

## 🔤 How an Address Looks

```
Sector  ·  Sub-Sector Word  ·  Local Word  ·  House No.
  14    ·       माया         ·    सूर्य     ·   0032
```

Four simple parts. Unique. Memorable. Shareable anywhere.
